# @vspo-lab/dayjs

A utility package for date manipulation and formatting using day.js.

## Installation

```bash
pnpm add @vspo-lab/dayjs
```

## Usage

```typescript
import { formatDate } from '@vspo-lab/dayjs';

// Example usage
const formattedDate = formatDate(new Date(), 'YYYY-MM-DD');
```

## Dependencies

- dayjs: ^1.11.13
- zod: ^3.24.3

## Development

```bash
# Build the package
pnpm build
```
